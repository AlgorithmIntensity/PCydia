# 📦 PCydia

PCydia — это простой менеджер пакетов для Windows, вдохновлённый идеологией Cydia для iOS. Позволяет легко устанавливать и управлять портативными программами из пользовательских репозиториев.

![PCydia](icon%20(1).png)

## ✨ Особенности

*   Установка портативных программ в один клик.
*   Поддержка пользовательских репозиториев.
*   Простой интерфейс с категориями и поиском.
*   Автоматическое создание ярлыков на рабочем столе (опционально).
*   Интеграция с файлами конфигурации `.pconf` для тонкой настройки установки.

## 🚀 Как это работает

1.  **Добавьте репозиторий:** Укажите URL репозитория (например, `https://ваш-пользователь.github.io/репозиторий/`) во вкладке "Источники".
2.  **Обновите список:** Нажмите кнопку "Обновить", чтобы загрузить актуальный список пакетов.
3.  **Выберите и установите:** Найдите нужную программу (доступны поиск и категории) и нажмите кнопку "Установить".
4.  **Готово!** Программа будет загружена и распакована в папку `%LOCALAPPDATA%\PCydia\apps\`.

## 🛠️ Сборка проекта

Проект написан на C# с использованием Windows Forms. Для сборки вам потребуется Visual Studio 2022 или новее.

**Необходимые компоненты:**
*   .NET Framework (версия указана в файле проекта `.csproj`)
*   Пакет NuGet: `SharpZipLib`

## 📁 Структура репозитория

*   `Form1.cs` — основной код программы.
*   `Program.cs` — точка входа.
*   `App.config` / `packages.config` — файлы конфигурации.
