📦 About PCydia

PCydia is a lightweight package manager for Windows inspired by Cydia for iOS. It allows you to easily install, update, and manage portable applications from custom repositories.

  • ✓ Simple and clean interface
  • ✓ Custom repositories support
  • ✓ Categories and search
  • ✓ Auto-install with .pconf files
  • ✓ Desktop shortcuts creation
  • ✓ Portable apps only (no registry)

🚀 Quick Start

  1. Download and run PCydia.exe
  2. Go to Sources tab → Add Source
  3. Enter repository URL
  4. Click Refresh
  5. Browse and install packages
📁 Apps installed to: %ProgramFiles%\AlgorithmIntensity\PCydia\apps\\
